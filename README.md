# Lonely City #

## Are you lonely? ##
Welcome to the Lonely City, a trivia-type game to determine if LC is the place for you.

* HTML/CSS
  * Bootstrap
* JavaScript/jQuery
* HTML DOM form

### Features include ###
1. Timer
   * Countdown till next question
1. Custom Ending
   * Two possible results dependent on responses.
   * Javascript logic.
